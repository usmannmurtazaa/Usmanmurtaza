## 🧑‍💻 Introduction

This is a fully responsive and animated portfolio website built using React.js and styled-components. It highlights my experience, technical skills, projects, and contact information in a modern and elegant layout.

---

## 🚀 Features

- **About Me**: Brief background and personal introduction.
- **Skills**: Technologies and tools I work with.
- **Projects**: A collection of my best work with live links and GitHub repositories.
- **Experience**: Professional work experience with role details and company info.
- **Education**: Academic background with timeline and achievements.
- **Resume**: Quick access to view or download my resume.
- **Contact**: A working contact form to get in touch directly.

---

## 🛠 Technologies Used

- **React.js** – Front-end framework
- **Styled-components** – For CSS-in-JS styling
- **HTML5 & CSS3** – Markup and structure
- **JavaScript (ES6+)** – Application logic
- **Framer Motion** – Animations and transitions
- **EmailJS** – For contact form integration
- **Git & GitHub** – Version control and code hosting
- **Netlify / Vercel** – Deployment and hosting
